# Desenvolvimento Web

Este repositório foi criado com o objetivo de armazenar os trabalhos, exercícios, desafios e projetos desenvolvidos durante as aulas de **Desenvolvimento Web**.

## Objetivo do repositório

O principal objetivo deste repositório é organizar e registrar a evolução dos estudos na área de desenvolvimento web, reunindo os conteúdos práticos realizados ao longo da disciplina.

Aqui serão armazenados arquivos relacionados a:

- Exercícios feitos em aula;
- Desafios propostos pelo professor;
- Projetos práticos;
- Atividades avaliativas;
- Testes e estudos com HTML, CSS e JavaScript.

## Tecnologias utilizadas

Durante o desenvolvimento dos trabalhos, poderão ser utilizadas as seguintes tecnologias:

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
